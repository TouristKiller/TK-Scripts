# TK ChordGun

Enhanced chord generator with scale filter/remap and chord recognition.
Based on pandabot ChordGun.

## Features
- Comprehensive scale library with 87 scales across 7 systems
- Circle of Fifths visualization
- Scale Filter and Remap modes
- Chord recognition system
- Automatic Voice Leading
- Arpeggiator
- Score and Guitar views
- And much more...

## Installation
Install via ReaPack.
